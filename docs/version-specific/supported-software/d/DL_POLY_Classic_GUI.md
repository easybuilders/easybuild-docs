# DL_POLY_Classic_GUI

The DL_POLY Graphical User Interface (or GUI) is a program written in the Java language and is intended for use with the DL_POLY molecular simulation program. This is the GUI for DL_POLY Classic, it can also be used for DL_POLY_4.

*homepage*: <https://gitlab.com/DL_POLY_Classic/dl_poly>
