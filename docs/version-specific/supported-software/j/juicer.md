# juicer

Juicer is a platform for analyzing kilobase resolution Hi-C data. This installation uses a hybrid setup: - Juicer Tools v2.20.00 (Standard) for CPU processing (pre, stats, pipeline). - Juicer Tools v3.0.0 (GPU) for accelerated Loop Calling (hiccups).

*homepage*: <https://github.com/aidenlab/juicer>
