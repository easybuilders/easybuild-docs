# LOBSTER

LOBSTER allows you to calculate projected COHP and COOP curves, and also reliable atom-projected DOS, directly based on plane-wave DFT output as given by the VASP or ABINIT or Quantum ESPRESSO packages.

*homepage*: <https://schmeling.ac.rwth-aachen.de/cohp/index.php?menuID=6>
