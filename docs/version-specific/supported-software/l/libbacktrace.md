# libbacktrace

A C library that may be linked into a C/C++ program to produce symbolic backtraces Initially written by Ian Lance Taylor iant@golang.org. This is version 1.0. It is likely that this will always be version 1.0.

*homepage*: <https://github.com/ianlancetaylor/libbacktrace/tree/master>
