# LAPACK++

LAPACK++ is a C++ wrapper around CPU and GPU LAPACK and LAPACK-like linear algebra libraries. BLAS++ is a C++ wrapper around CPU and GPU BLAS (basic linear algebra subroutines.

*homepage*: <https://github.com/icl-utk-edu/lapackpp>
