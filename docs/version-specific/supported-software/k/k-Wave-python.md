# k-Wave-python

This project is a Python implementation of v1.4.0 of the MATLAB toolbox k-Wave: http://www.k-wave.org/ as well as an interface to the pre-compiled v1.3 of k-Wave simulation binaries, which support NVIDIA sm 5.0 (Maxwell) to sm 9.0a (Hopper) GPUs.

*homepage*: <http://www.k-wave.org/download.php>
