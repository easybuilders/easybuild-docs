# ffnvcodec

FFmpeg nvidia headers. Adds support for nvenc and nvdec. Requires Nvidia GPU and drivers to be present (picked up dynamically).

*homepage*: <https://git.videolan.org/?p=ffmpeg/nv-codec-headers.git>
