# FAAD2

FAAD2 is a HE, LC, MAIN and LTP profile, MPEG2 and MPEG-4 AAC decoder. FAAD2 includes code for SBR (HE AAC) decoding.

*homepage*: <https://github.com/knik0/faad2>
