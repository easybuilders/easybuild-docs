# FUNAERO

Free-wake UNsteady AEROdynamics (FUNAERO) code features an aerodynamic formulation based on the Boundary Element Method (BEM) to predict the behaviour of a wind turbine operating in a spatially uniform/nonuniform inflow.

*homepage*: <https://www.inm.cnr.it>
