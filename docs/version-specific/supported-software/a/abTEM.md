# abTEM

abTEM is a flexible open-source package for simulating transmission electron microscopy experiments. We strive for easy integration with other simulation codes and analysis tools accessible from Python. Utilization of other open-source projects makes abTEM fast and scalable from a laptop to a high-performance computing environment.  abTEM supports GPU acceleration if the CuPy module is also loaded.

*homepage*: <https://abtem.readthedocs.io>
