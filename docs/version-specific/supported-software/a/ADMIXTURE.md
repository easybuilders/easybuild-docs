# ADMIXTURE

Software tool for maximum likelihood estimation of individual ancestries from multilocus SNP genotype datasets. It uses the same statistical model as STRUCTURE but calculates estimates much more rapidly using a fast numerical optimization algorithm.

*homepage*: <https://dalexander.github.io/admixture/index.html>
