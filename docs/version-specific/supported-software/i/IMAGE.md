# IMAGE

IMAGE (Motif  Activity and  Gene  Expression changes of transcription factors) is software for integrated analysis of motif activity and gene expression changes of transcription factors. IMAGE makes prediction of causal transcription factors based on transcriptome profiling and genome-wide maps of enhancer activity.

*homepage*: <https://github.com/JesperGrud/IMAGE>
