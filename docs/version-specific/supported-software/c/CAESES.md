# CAESES

CAESES stands for "CAE System Empowering Simulation" and its ultimate goal is to help engineers to design optimal flow-exposed products. It provides simulation-ready parametric CAD for complex free form surfaces, and targets CFD-driven design processes.

*homepage*: <https://www.caeses.com>
