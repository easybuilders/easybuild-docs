# cuSOLVERMp

NVIDIA cuSOLVERMp is a high-performance, distributed-memory, GPU-accelerated library that provides tools for the solution of dense linear systems and eigenvalue problems.  cuSOLVERMp is compatible with 2D block-cyclic data layout and provides ScaLAPACK-like C APIs.

*homepage*: <https://docs.nvidia.com/cuda/cusolvermp/>
