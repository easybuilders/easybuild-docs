# SYSMOIC

SYSMOIC is a research, non-commercial, program package for the calculation and graphical visualization of a number of time-independent (static) and time-dependent (dynamic) magnetic and electric molecular property densities, which are invariant with respect to the choice of the reference origin.

*homepage*: <https://sysmoic.chem.unisa.it/MANUAL/>
