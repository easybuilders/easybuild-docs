# SEACR

SEACR: Sparse Enrichment Analysis for CUT&RUN SEACR is intended to call peaks and enriched regions from sparse CUT&RUN or chromatin profiling data in which background is dominated by "zeroes" (i.e. regions with no read coverage).

*homepage*: <https://github.com/FredHutch/SEACR>
