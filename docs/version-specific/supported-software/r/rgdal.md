# rgdal

Provides bindings to the 'Geospatial' Data Abstraction Library ('GDAL') (>= 1.11.4 and <= 2.5.0) and  access to projection/transformation operations from the 'PROJ.4' library.

*homepage*: <http://rgdal.r-forge.r-project.org/>
