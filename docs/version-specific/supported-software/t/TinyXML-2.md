# TinyXML-2

TinyXML-2 is a simple, small, efficient, C++ XML parser that can be easily integrated into other programs. It parses an XML document, and builds from that a Document Object Model (DOM) that can be read, modified, and saved.

*homepage*: <http://leethomason.github.io/tinyxml2>
