# PyBEL

PyBEL is a pure Python software package that parses BEL (Biological Expression Language) documents, validates their semantics, and facilitates data interchange between common formats and database systems like JSON, CSV, Excel, SQL, CX, and Neo4J.

*homepage*: <https://github.com/pybel/pybel>
