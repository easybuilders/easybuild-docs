# ParaView-Catalyst

ParaView Catalyst provides a small, easy-to-use, API that any simulation developed in C++, C, Fortran or Python can use to do in situ analysis without developing its own custom data analysis code.

*homepage*: <https://docs.paraview.org/en/latest/Catalyst/>
